### 对比度优化措施

1. **前景色优化**:
   ```css
   --foreground: 210 40% 98%; /* 从213 31% 91%优化 */
   ```
   - 提高亮度从91%到98%
   - 显著增强与背景的对比度

2. **静音前景色优化**:
   ```css
   --muted-foreground: 215 25% 85%; /* 从20% 65%优化 */
   ```
   - 提高亮度从65%到85%
   - 确保次要文本仍然可读

3. **边框对比度增强**:
   ```css
   --border: 216 34% 20%; /* 从17%提高到20% */
   ```
   - 增强边框可见性
   - 新增`--border-strong: 217° 33% 28%`用于重要边框

4. **文本阴影系统**:
   ```css
   --text-shadow: 0 1px 2px rgb(0 0 0 / 0.3);
   ```
   - 为所有高对比度文本添加阴影

## 🛠️ 高对比度工具类

```css
/* 高对比度文本工具类 */
.text-high-contrast {
  color: hsl(var(--foreground));
  text-shadow: var(--text-shadow);
}

/* 高对比度次要文本 */
.text-high-contrast-secondary {
  color: hsl(var(--muted-foreground));
  text-shadow: var(--text-shadow);
}

/* 图标增强对比度 */
.icon-high-contrast {
  color: hsl(var(--muted-foreground));
  filter: drop-shadow(0 1px 1px rgb(0 0 0 / 0.5));
}
```

- ✅ 所有文本都有适当的阴影增强
